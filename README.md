# 💻 Smart Code Reviewer

A web-based tool that analyzes source code and provides insights on time complexity and performance.

## 🚀 Features
- Detects loops and estimates time complexity
- Identifies nested loops and inefficient patterns
- Provides optimization suggestions
- Detects recursion and long code

## 🛠 Tech Stack
- Python
- Streamlit

## ▶️ Run Locally
```bash
pip install streamlit
python -m streamlit run app.py
